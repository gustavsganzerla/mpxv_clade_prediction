Web Server Instructions

Upload files

Drag/drop or manually upload your .fasta file containing your sequences. Multiple files are accepted.

Copy the text content of your files

Copy all the content of your .fasta file containing your sequences and paste it to the Genome text box.

Predict

Click on the Predict button. You will be directed to a webpage containing the results.
Docker Usage Instructions

Pull the Docker image:

docker pull sganzg/ai-clade:latest

Run with a directory of FASTA files:

docker run --rm -v /local/path/to/fasta:/data sganzg/ai-clade:latest /data

Run with output directory:

docker run --rm -v /local/path/to/fasta:/data -v /local/path/to/results:/results sganzg/ai-clade:latest /data --output /results

Check results:

/local/path/to/results/results.csv
